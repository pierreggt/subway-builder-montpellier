Montpellier | Subway Builder

🇬🇧 English | 🇫🇷 Français

🇬🇧 English

Subway Builder mod adding a map of the Montpellier metropolitan area (France), from Sète and the Mediterranean coast to Lunel and the Hérault garrigue.

✨ Features


🏙️ 456,000 buildings from OpenStreetMap (binary format)
👥 650,000+ simulated commuters from real INSEE census data (Filosofi 2021, mobility flows 2021)
🎓 Realistic traffic generators: Gare Saint-Roch, CHU Lapeyronie hospital, university campuses (Triolet, Saint-Priest, Faculty of Medicine), Odysseum, Millénaire
✈️ Montpellier-Méditerranée airport with runways and taxiways
🌊 Ocean depth data — the Mediterranean is fully modelled with depth contours
🏛️ Building & ocean foundations layers enabled
🗺️ Protomaps basemap with water, parks and forests


📦 Installation


In Subway Builder: Settings → Mod Manager → Open Mods Folder
Unzip and place the Montpellier folder inside mods
Start the tile server:

macOS/Linux: open Terminal, type sh  (with a space), drag and drop run_mac-linux.sh, press Enter
Windows: double-click run_windows.bat



Enable the mod in Mod Manager, restart the game
Main menu: Ctrl + Shift + R (Cmd + Option + R on Mac)
New Game → France tab → Montpellier


⚠️ Keep the terminal window open while playing.

🇫🇷 Français

Mod Subway Builder ajoutant la métropole montpelliéraine, de Sète et la côte méditerranéenne jusqu'à Lunel et la garrigue héraultaise.

✨ Contenu


🏙️ 456 000 bâtiments OpenStreetMap (format binaire)
👥 Plus de 650 000 actifs simulés (INSEE Filosofi 2021, mobilités 2021)
🎓 Pôles de trafic : Gare Saint-Roch, CHU Lapeyronie, campus Triolet et Saint-Priest, Faculté de Médecine, Odysseum, Millénaire
✈️ Aéroport de Montpellier-Méditerranée avec pistes et taxiways
🌊 Données de profondeur de la Méditerranée (contours bathymétriques)
🏛️ Couches fondations bâtiments et océan activées
🗺️ Fond de carte Protomaps avec eau, parcs et forêts


📦 Installation


Dans Subway Builder : Paramètres → Mod Manager → Open Mods Folder
Décompresser et placer le dossier Montpellier dans mods
Lancer le serveur : sh  + glisser run_mac-linux.sh dans le Terminal, Entrée (ou double-clic sur run_windows.bat)
Activer le mod, redémarrer le jeu
Menu principal : Ctrl + Shift + R (Cmd + Option + R sur Mac)
Nouvelle partie → onglet France → Montpellier
