# Montpellier | Subway Builder
 
[🇬🇧 English](#-english) | [🇫🇷 Français](#-français)
 
## 🇬🇧 English
 
Subway Builder mod adding a map of the **Montpellier-Méditerranée area** (France), from Sète and the Mediterranean coast to Nîmes, Arles and the Camargue.
 
### ✨ Features
 
- 🏙️ **456,000 buildings** from OpenStreetMap (binary format)
- 👥 **750,000+ simulated commuters** from real INSEE census data (Filosofi 2021, mobility flows 2021)
- 🎓 **Realistic traffic generators**: Gare Saint-Roch, Gare Sud-de-France, CHU Lapeyronie, CHU Carémeau (Nîmes), university campuses (Triolet, Saint-Priest, Faculty of Medicine), Odysseum, Millénaire, Sud de France Arena
- ✈️ **Montpellier-Méditerranée airport** with runways and taxiways
- 🌊 **Ocean depth data** — the Mediterranean is fully modelled with depth contours
- 🏛️ **Building & ocean foundations** layers enabled
- 🗺️ **Protomaps basemap** with water, parks and forests
- 🏖️ Full coverage: Montpellier, Nîmes, Arles, Sète, Lunel, La Grande-Motte, Aigues-Mortes and the coast
### 📦 Installation via Railyard (recommended)
 
Install [Railyard](https://subwaybuildermodded.com), search for **Montpellier** and click Install. No terminal needed.
 
### 📦 Manual installation
 
1. Download `montpellier_v1.0.0.zip` from the [latest release](../../releases/latest)
2. In Subway Builder: Settings → Mod Manager → **Open Mods Folder**
3. Unzip and place the `Montpellier` folder inside `mods`
4. Start the tile server:
   - **macOS/Linux**: open Terminal, type `sh ` (with a space), drag and drop `run_mac-linux.sh`, press Enter
   - **Windows**: double-click `run_windows.bat`
5. Enable the mod in Mod Manager, restart the game
6. Main menu: **Ctrl + Shift + R** (Cmd + Option + R on Mac)
7. New Game → **France** tab → **Montpellier**
⚠️ For manual install: keep the terminal window open while playing.
